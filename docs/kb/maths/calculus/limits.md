# Limits

## Definition

Limit describes behavior of a function as input approaches a value.

lim x→a f(x)

## Standard Limits

lim x→0 (sin x / x) = 1

lim x→0 (1 − cos x)/x² = ½

lim x→0 (e^x −1)/x = 1

## Important Techniques

1. Factorization
2. Rationalization
3. Standard limits
4. L'Hospital rule

## L'Hospital Rule

If limit gives 0/0 or ∞/∞:

lim f/g = lim f'/g'
